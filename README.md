# PDF Merger & Splitter Utility

A handy Python Command Line Interface (CLI) utility tool that allows users to seamlessly combine multiple PDF documents into a single file or extract pages from a single PDF into individual documents.

## ✨ Features
- **PDF Merging:** Combine two or more PDF files in a specific order into a single unified file.
- **PDF Splitting:** Automatically extracts every single page of a multi-page PDF and saves them as separate files.
- **Dynamic File Input:** Direct entry handling for processing multiple files via terminal inputs.
- **Input Validation:** Safe path checks that prevent code crashes if file names are mistyped.

## 🚀 Tech Stack Used
- **Language:** Python 3.x
- **Core Library:** `pypdf` (Powerful open-source pure-python PDF library)

## 💻 How to Run the Application

1. **Install Dependencies:**
   Install the required pypdf package via pip:
   ```bash
   pip install pypdf
   ```

2. **Run the Script:**
   Execute this command inside your terminal:
   ```bash
   python pdf_util.py
   ```
